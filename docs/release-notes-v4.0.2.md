ServicePilot 4.0.2 是一次修复版本：解决折叠组在日志持续输出时被重新弹开的问题。

## 修复

- **折叠组被日志刷新重新弹开**：日志持续输出时，手动关闭一个折叠组后，若该组随后又有新行输出，折叠状态会被错误地重新展开。根因是增量重建折叠时 AvalonEdit 可能销毁并重建折叠区，导致用户手动设置的折叠状态丢失。现在按「组头」记录折叠意图，并在每次重建后回写，手动折叠/展开状态在持续输出和切换标签时都能稳定保持。

## 要求

- Windows
- 使用发布页自包含 `ServicePilot.exe` 时不需要单独安装 .NET 运行时。

## 安全说明

- ServicePilot 会执行用户配置的本地脚本；合并脚本同样是用户提供的 C# 代码，运行在应用进程内，请只使用你信任的合并脚本。
- AI 帮助内容会包含本机 exe 绝对路径，只建议发送给可信个人 AI 助手。
