ServicePilot 4.0.0 是一次重大功能版本：引入可编程的**日志合并脚本引擎**与 VSCode 式日志折叠、右侧颜色概览导航、外部配置文件热加载，并全面刷新品牌图标。

## 亮点

- **日志合并脚本引擎（Log Merge Script）**：步骤/模板动作支持自定义 C# 合并函数，可将多行日志按规则折叠为一行摘要并着色。脚本入参新增 `PreviousResult`、`PreviousWasCollapsed`、`InCollapseGroup`，返回结果新增 `State` 字典用于跨行携带状态（仅运行期，不持久化）；服务/模板编辑页新增合并函数代码框并预填带注释的模板。CLI 提供 `merge-script test` 验证。
- **VSCode 式日志折叠**：保留原始行的同时生成可展开折叠组，折叠时只显示摘要；搜索命中折叠内容会自动展开；新增“折叠全部/展开全部”摘要视图开关。折叠占位文字为白色，左侧加号与文本之间用约 100px 色块显示该组颜色。
- **右侧颜色概览导航**：日志右侧新增静态可点击颜色概览图，按优先级聚合每行颜色（错误 > 警告 > 自定义 > 系统 > 普通），点击可快速跳转；感知折叠状态，仅反映可见日志分布。
- **外部配置热加载**：直接编辑 `%APPDATA%\ServicePilot\config.v2.json` 后托盘自动检测并热加载，不再覆盖外部改动，保留运行中服务的运行状态。
- **菜单可滚动 + 系统主题色**：托盘各级菜单在项目过多时可滚动并统一样式；服务管理选中条改用 Windows 系统主题色。
- **全新应用图标**：采用更精致的青色透明圆角图标，同步应用到 exe、任务栏和各窗口标题栏，消除旧图标白边。
- **日志窗口标题栏**：完整标题移入标题栏；`ai-help` 补回完整 CLI 帮助。

## 下载

- `ServicePilot.exe`

## 要求

- Windows
- 使用发布页自包含 `ServicePilot.exe` 时不需要单独安装 .NET 运行时。

## 安全说明

- ServicePilot 会执行用户配置的本地脚本；合并脚本同样是用户提供的 C# 代码，运行在应用进程内，请只使用你信任的合并脚本。
- AI 帮助内容会包含本机 exe 绝对路径，只建议发送给可信个人 AI 助手，不要公开贴到网页、Issue 或日志。
- 自动化测试请设置 `SERVICEPILOT_CONFIG_DIR`，避免误改真实 `%APPDATA%\\ServicePilot` 配置。
