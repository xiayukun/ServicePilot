## README 首屏卖点

- AI 可以先读 `ServicePilot.exe ai-help` 和 `ServicePilot.exe doctor --json`，再用 JSON 状态安全操作。
- 托盘数字就是当前活跃服务数，适合长期挂在任务栏。
- 支持多项目、多目录、多脚本步骤。
- 支持启动变量和手动步骤变量，适合切换后端地址、环境名、配置值。
- 不安装系统服务，不要求管理员权限，更适合本地开发。
- Windows Job Object 清理进程组，解决 npm/Vite 停止后端口残留问题。
