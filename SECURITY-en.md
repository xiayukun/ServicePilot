# Security Policy

中文：[SECURITY.md](SECURITY.md)

ServicePilot manages process execution and script running, so security reports are welcome.

## Supported Versions

Only the latest release is supported.

## Reporting a Vulnerability

Please open a private security advisory on GitHub if the repository is public, or contact the maintainer directly before publishing details.

Useful report details:

- Windows version
- ServicePilot version
- steps to reproduce
- whether the GUI or tray menu was used

## Safety Expectations

ServicePilot executes user-configured scripts. Reports about command injection, privilege escalation, or unintended process execution are treated as high priority.
